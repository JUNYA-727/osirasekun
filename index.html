アプリの画面

Features

主な機能

めんどくさいをなくして、やりたいことだけに集中。

💬

自然言語でかんたん入力

日本語でも英語でもOK。「明後日の午後3時に歯医者」「next Friday 9am standup」みたいに、いつもの言葉でそのまま。

🔢

数字だけでサクッと登録

「0315 1400 歯医者」みたいに数字で一発。範囲指定も「03150900 03151700 会議」で。

⏱

「〜分後」でさっと登録

「30分後 薬飲む」「2時間後 ミーティング」。今からの時間でもそのまま登録できる。

📅

4つのカレンダービュー

リスト・日・週・月をワンタップで切替。Googleカレンダーみたいな画面で予定がすぐわかる。

🔔

通知でうっかり防止

タスクごとにリマインダーをセット。16種類の通知メッセージから選べる。終日タスクにも対応。

🔄

繰り返し＆習慣トラッキング

毎日・毎週・毎月の繰り返しに対応。連続日数や達成回数もチェックできる。

🔒

完全オフライン対応

処理は全部端末の中で完結。データが外に出ることは一切なし。ネットなしでも使える。

🎨

自分好みにカスタマイズ

「朝」「夜」の時刻変更、自分だけの辞書、8種類のアイコン、ライト/ダーク/システム連動テーマ。

Input Examples

入力例

こう打つだけ。あとはおしらせくんがやってくれる。

明日の夜ミーティング → 明日 20:00
30分後 薬飲む → 30分後
来週月曜 朝 ジョギング → 来週月曜 9:00
9時から17時まで仕事 → 9:00 - 17:00
20260315 1400 歯医者 → 2026/3/15 14:00
0401 0405 旅行 → 4/1 - 4/5 終日

まずは使ってみてください

思いついたことを、そのまま打ち込むだけ。

Privacy Policy

プライバシーポリシー

はじめに

おしらせくん（以下「本アプリ」）は、ユーザーのプライバシーを大切にしています。このページでは本アプリでの情報の扱いについて説明します。

データの収集

本アプリは個人情報を一切収集しません。タスク・スケジュール・設定などのデータはすべて端末内にのみ保存され、外部サーバーに送信されることはありません。

データの処理

自然言語の解析を含むすべての処理は端末上で実行されます。インターネット接続は不要で、オフラインでもすべての機能を利用できます。

第三者への提供

ユーザーのデータを第三者に提供・販売・共有することはありません。

分析ツール・広告

サードパーティの分析ツールや広告SDKは使用していません。

ポリシーの変更

本ポリシーは必要に応じて更新する場合があります。変更がある場合はこのページでお知らせします。
